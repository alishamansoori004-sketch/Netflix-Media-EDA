# 🎬 Netflix Media EDA

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset. The objective is to discover trends, patterns, and business insights using Python and data visualization techniques.

---

## 🎯 Objectives

- Analyze Movies vs TV Shows distribution
- Explore release year trends
- Identify top countries producing Netflix content
- Find the most common genres
- Analyze content ratings
- Explore top directors and actors
- Study movie duration distribution
- Analyze yearly content additions

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Plotly
- Jupyter Notebook

---

## 📂 Dataset

Netflix Movies and TV Shows Dataset

Columns include:

- Title
- Type
- Director
- Cast
- Country
- Release Year
- Rating
- Duration
- Genre
- Description

---

## 📊 Key Insights

- Movies dominate Netflix's content library.
- Most content was added between 2016–2020.
- The United States contributes the highest number of titles.
- Drama and International Movies are the most common genres.
- TV-MA and TV-14 are the most common content ratings.
- Most movies have durations between 80–120 minutes.
